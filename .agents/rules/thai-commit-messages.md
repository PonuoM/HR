---
name: การเขียนข้อความ Git Commit เป็นภาษาไทย
description: บังคับให้ต้องมีคำอธิบายภาษาไทยในข้อความ Git commit เสมอ
---

เมื่อทำการบันทึกโค้ด (commit) ผ่านระบบ Git คุณจะต้องแนบคำอธิบายเป็น "ภาษาไทย" ไว้ในข้อความ commit เสมอ

* สามารถใช้รูปแบบมาตรฐาน Conventional Commits (เช่น feat:, fix:) ในส่วนต้นได้ แต่จะต้องมีคำอธิบายหรือสรุปการทำงานเป็นภาษาไทยต่อท้าย หรืออยู่ในเนื้อหา (body) ของ commit นั้นๆ 
* ตัวอย่าง: `git commit -m "feat: add searchable dropdown (เพิ่มระบบค้นหาใน Dropdown)"`
